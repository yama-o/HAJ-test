# HARUKI ANIME JOURNAL - Landing Page

サイバーパンクな雰囲気とモダンな洗練さを融合させた、架空のアニメ情報メディア「HARUKI ANIME JOURNAL」のランディングページ実装テンプレートです。

## 📖 概要 (Overview)

提供されたデザインイメージを基に、HTML5とCSS3のみで構築されています。
ネオンカラー（シアン・オレンジ）のアクセントとダークモードを基調とし、没入感のあるUIを再現しました。

## ✨ 特徴 (Features)

* **レスポンシブデザイン:** PC、タブレット、スマートフォンに対応。
* **Pure CSS:** フレームワーク（Bootstrap等）やJavaScriptを使用せず、軽量なコードで実装。
* **モダンなエフェクト:**
    * カード要素のホバー時の浮き上がり＆発光演出。
    * 背景や枠線へのネオンカラー（`box-shadow`）の活用。
* **カスタマイズ性:** CSS変数（Custom Properties）を使用しているため、配色の変更が容易。
